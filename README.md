# psd_deploy_uas

## Model ini dilatih menggunakan dataset publik:
[UCR Strawberry Dataset](https://www.timeseriesclassification.com/description.php?Dataset=Strawberry)

## Cara Menjalankan

1. Install dependencies

```pip install -r requirements.txt```

2. Jalankan aplikasi

```streamlit run app.py```

Hasil Website:
[Sistem Deteksi Keaslian Stroberi](https://psddeployuas.streamlit.app/)
